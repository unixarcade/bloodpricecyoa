## ✦ Features

### ❧ Single-File GitHub Pages Build

The whole adventure is designed to run from a single HTML file. It can be hosted on GitHub Pages, opened locally, archived, shared, copied, studied, and preserved.

### ❧ Cinematic Manuscript Interface

The game presents itself as a living folio: warm paper, dark sanctuary UI, gold ornament, red shadow, illuminated glyphs, page turns, sigils, and a sacred book atmosphere.

### ❧ Procedural Route Weather

Each route breathes with changing atmospheric pressure. The page includes procedural route pulses and weather-like mythic signals, giving replayed journeys a living texture.

### ❧ Moral Consequence Engine

Choices carry weight. Frank’s actions affect the route through visible and hidden state.

The path tracks five core pressures:

```text
W — Wisdom
S — Shadow
M — Mercy
H — Human
V — Veil
```

The route is not merely read.
It is accumulated.

### ❧ Replayable Routes

The game supports fresh routes, seeded variation, opened-door memory, backtracking, endings, and route review. Each run may feel like another pass through the same sacred trial under different light.

### ❧ Route Index

The index reveals every door Frank has seen. Unseen folios remain sealed until the player reaches them.

### ❧ Sound, Reading, and Controls

Controls:

```text
Tap / Click choices     Choose a route
Number keys             Select choices
Space                   Read
M                       Toggle sound
I                       Open index
Back                    Return one folio
Save                    Preserve progress
Reset                   Begin again
```

### ❧ Oracle, Roll, Save, Reset

The side panel includes ritual-game controls for oracle consultation, rolling, saving, and resetting the route.

### ❧ Mobile-Safe Sanctuary Layout

The interface is built so mobile stats and choices remain in the page flow rather than being hidden behind overlapping controls. It is meant to work on both phones and laptops.

## ✦ How to Run Locally

Clone the repository:

```bash
git clone https://github.com/unixarcade/bloodpricecyoa.git
cd bloodpricecyoa
```

Open the file directly:

```text
index.html
```

Or serve it locally:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
